-----------------------------------

-----------------------------------
DESIGN STYLE:
-----------------------------------

- Rounded corners (16–20px)
- Soft shadow
- Clean typography (Poppins)
- Minimal text, strong CTA button
- Add subtle icon or illustration (optional)

PLACEMENT:
-----------------------------------
Home screen:
Place banner BELOW primary actions and ABOVE QR profiles section.

4. SAFETY FEATURE (DEFAULT)

Inside component:
- Determine which banner to show
