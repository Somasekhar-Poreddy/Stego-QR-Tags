Build a full-featured Super Admin Panel for Stegofy using React + Tailwind + Supabase.

-----------------------------------
GOAL:
-----------------------------------
Create a centralized control system to manage:

- Users
- QR Codes
- Contact Requests
- Products & Shop
- Orders
- QR Inventory (pre-generated)
- Admin roles & permissions
- Analytics
- Activity logs
- Notifications
- Support system

-----------------------------------
TECH:
-----------------------------------
- React (functional components)
- Tailwind CSS
- Supabase (DB + Auth)
- Use existing Supabase client
- Use modular architecture

-----------------------------------
LAYOUT:
-----------------------------------
- Left Sidebar Navigation
- Top Header (search + admin profile)
- Main content area

-----------------------------------

1. Dashboard
2. Users
