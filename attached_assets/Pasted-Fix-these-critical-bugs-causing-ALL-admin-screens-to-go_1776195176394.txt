FIX 1 — MOST CRITICAL
File: artifacts/stegofy-qr/src/admin/AdminRouter.tsx

Find this line:
  <Switch key={refreshKey}>

Change it to:
  <Switch>

Then find the variable declarations for useSessionKeepalive:
  const { sessionOk, reconnecting, refreshKey } = useSessionKeepalive();

Change it to:
  const { sessionOk, reconnecting } = useSessionKeepalive();

REASON: key={refreshKey} unmounts and remounts every screen on every token
refresh, which happens on every tab switch. This is the main cause of all
screens going blank.

  ensureFreshSession()
    .then(() => ...)
    .catch(() => { setXxxError(true); })

Change the .catch() to NOT set an error on AuthExpiredError specifically.
Instead use this pattern:

  ensureFreshSession()
    .then(() => getDashboardStats())  
    .then((data) => { setStats(data); setStatsError(false); })
    .catch((e) => {
      // Only show error for non-auth failures
      // Auth failures are handled globally by useSessionKeepalive

        setStatsError(true);
      }
    })
    .finally(() => setStatsLoading(false));
