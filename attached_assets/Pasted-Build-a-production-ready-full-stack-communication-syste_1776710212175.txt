Build a production-ready full-stack communication system for a QR-based contact platform (Stegofy).

STACK:
- Frontend: React (modern UI, Tailwind)
- Backend: Node.js (Express)
- Database: Supabase
- Providers:
  - Zavu (Primary WhatsApp)
  - Exotel (Calls + SMS + WhatsApp fallback)

--------------------------------------------------

CORE ARCHITECTURE:

Create a "Communication Router Service" in backend that handles all outbound communication.

Routing priority:
1. Zavu WhatsApp (primary)
2. Exotel WhatsApp (fallback)
3. Exotel SMS (final fallback)
4. Exotel Calls (only when triggered)

--------------------------------------------------

FEATURE 1: OTP SYSTEM (WHATSAPP FIRST + FALLBACK)

Flow:
1. User enters phone number
2. Backend generates OTP (store in Supabase with expiry)
3. Send OTP via Zavu WhatsApp
4. Start 10–15 second timer
5. If:
   - delivery failed OR timeout:
        → send via Exotel WhatsApp
6. If still failed:
        → send SMS via Exotel
7. User submits OTP → verify from DB

UI:
- Show "Sending via WhatsApp..."
- Timer (15 sec)
- Button: "Send via SMS"

--------------------------------------------------

FEATURE 2: CONTACT REQUEST FLOW

1. User scans QR
2. Select intent
3. Send message via WhatsApp (Zavu)
4. If fails → Exotel WhatsApp → SMS fallback

NO approval system required.

--------------------------------------------------

FEATURE 3: DIRECT CALL SYSTEM (EXOTEL)

Flow:
1. User enters phone + PIN
2. Click "Call Owner"
3. Trigger Exotel call immediately

Constraints:
- Max call duration: configurable (default 60 sec)
- Auto disconnect after limit
- Rate limit:
    - Max 2 calls per QR per hour
    - Cooldown: 60 seconds

Server-side enforcement REQUIRED even if UI hides button.

--------------------------------------------------

FEATURE 4: MULTI-PROVIDER FAILOVER

Implement robust failover:

Function: sendMessageSmart()

Logic:
- Try Zavu
- If failed OR timeout → Exotel WhatsApp
- If failed → SMS

Use webhook-based delivery tracking:
- Zavu webhook
- Exotel webhook

Store message status in DB.

--------------------------------------------------
